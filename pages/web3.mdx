# Web3学习笔记 
###### <p align="right">———— 源自[《一个Web3开发的入门历程》](https://juejin.cn/post/7138072108516507661)  , [《2022年-Web3开发者学习路线指南》](https://juejin.cn/post/7114306640760799263)</p>

### Web3.0概念

Web 1.0 -> Web 2.0 -> Web 3.0: 不断进化
<Image src={WEB3_IMG1} width={800} height={300} />

Web 1.0——网站投喂信息给用户, 缓慢的, 完全受控制的。

Web 2.0——互联网开始与人互动, 主要目标是，使互联网更加民主，并尽可能地方便用户访问。

Web 3.0——更通人性的, 由人来控制的互联网。

Web3是一个包含了很多领域的概念,不讨论币圈和链圈的划分,Web3包括有Defi、NFT、Game等基于区块链的Dapp应用的开发；也有VR、AR等追求视觉沉浸感的XR相关领域的开发；还有基于区块链底层架构或者协议的开发。

### Web3.0的特点

1、统一身份认证系统

2、数据确权与授权

3、隐私保护与抗审查

4、去中心化运行

隐私、数据、权利、审查、身份……这都是区块链 Web 3.0 关心的话题。
<Image src={WEB3_IMG2} width={800} height={300} />

### Web3.0的现状

Dapp的生态 [dappradar.com](https://dappradar.com)

MetaMask（最知名的区块链钱包），Stepn（一个能购买跑鞋，跑步会赚取收益的Dapp）、Axie Infinity（一款神奇宝贝主题的区块链游戏）、NBA TopShot（NBA官方验证的NFT）、OpenSea（全球最大的NFT交易市场）等等，你还会发现如今区块链生态的繁荣，目前有很多的Dapp不是在以太坊这一条链上运行的了，还有BNB Chain、SOLONA、TRON等众多公链生态。

火热发展的同时也存在很多问题。


- Web 3.0 效率低、普及难度大，去中心化网络对于任务处理的速度低下。
- Web 3.0 要对用户科普的内容要远远超过区块链、比特币、加密货币要科普的知识。当然，开发者也可以通过多样的设计将应用程序设计的和传统 App 无异，让用户根本感觉不到这些复杂的运行机制的存在。但这类开发工具，才刚刚起步
- 通证经济和人性自私，炒币远比做项目赚钱。

### Web 3.0 是否值得存在

从去中心化以及个人权益保护的角度来看，Web 3.0 有它的价值。

作为技术人员，不管是噱头还是商业炒作，我们更应该关注的是他的技术层面。
